# Tic-Tac-Toe (C++)

## Overview
This project is a terminal-based Tic-Tac-Toe game written in C++. It allows two players to compete by taking turns placing their marks on a 3x3 board. The program checks for valid moves, detects winners, and determines when the game ends in a draw.

This project demonstrates core programming concepts such as loops, conditionals, functions, arrays, input validation, and game logic.

---

## Purpose
The purpose of this project was to practice building an interactive console application while applying problem-solving skills and structured program design in C++.

It also helped strengthen understanding of:
- User input handling
- Conditional logic
- Repetition with loops
- Board state management
- Basic algorithm design

- Standard Library (`iostream`, etc.)
